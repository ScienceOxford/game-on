Make the Machine Teleport

You could make the game more difficult by stopping the player from getting too close to the machine.

This is the most complex of the extension instructions, so consider trying one of the other ideas first.

To start with, let's make the player lose a life if they overlap the machine!

You already have an overlap loop for this, so find this block and add a change life by -1 block from the Info menu to it.


Test out this code by moving your player over the machine.

Your player loses all three lives, and the game ends! This is because the program runs very quickly, and you cannot move the player away quick enough to lose only one life.

To fix this, we can destroy the machine sprite, and recreate it somewhere else on the screen.

Replace the otherSprite start effect block with the destroy mySprite block. Remember to change this block so that it effects the otherSprite, and use the plus button to add your effect.

Next, recreate and position the machine sprite, using the same code you used in your on start loop - this time put the machine in a different position.


Test it out to make sure that your machine moves.
